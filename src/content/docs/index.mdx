## 🤝 함께하기

Sudoremove는 오픈 커뮤니티입니다. 다음과 같은 방법으로 참여할 수 있습니다:

- **문서 개선**: 오타 수정부터 새로운 튜토리얼 작성까지
- **코드 기여**: 예제 코드와 프로젝트 공유
- **커뮤니티 활동**: 질문 답변, 지식 공유
- **피드백**: 개선 사항 제안

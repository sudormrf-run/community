# Koch v1.1

Koch v1.1은 Alexander Koch의 오리지널 저비용 로봇 디자인을 개선한 버전입니다. 연구와 학습을 위한 경제적이고 모듈형 설계의 로봇 팔로, LeRobot과 완벽하게 호환됩니다.

<Callout type="tip" title="두 가지 버전">
  Koch 로봇에는 두 가지 버전이 있습니다:
  - **오리지널**: Alexander Koch의 첫 번째 디자인
  - **v1.1**: Jess Moss의 개선된 버전
</Callout>

## 🔧 주요 개선사항 (v1.1)

### 하드웨어 개선
- 나사 간섭 문제 해결
- 표준화된 구멍 크기
- 플라스틱에 직접 나사를 고정하는 방식 제거
- 리더 로봇에 보드 플랫폼 추가
- 팔로워 로봇의 바닥 물체 집기 플랫폼 추가

### 제작 편의성
- **납땜 불필요**: 모든 연결이 커넥터로 가능
- **SolidWorks 모델 제공**: CAD 파일로 수정 가능
- **배선 다이어그램**: 명확한 연결 가이드
- **조립 영상**: 애니메이션 포함 상세 가이드

## 🛠️ 제작 가이드

### 3D 프린팅 설정

```
• 레이어 높이: 0.2mm
• 노즐 직경: 0.4mm (최대)
• 인필 밀도: ~30%
• 권장 소재: PLA+, ABS, PETG
```

### 전원 요구사항

<Callout type="warning">
  전원 공급 전압에 주의하세요:
  - **리더 암**: 5V
  - **팔로워 암**: 12V
</Callout>

### 연결
- 두 암은 USB-C로 컴퓨터에 연결
- 각 암은 독립적인 전원 공급 필요

## 🚀 소프트웨어 설정

### LeRobot 통합

```bash
# LeRobot 설치
pip install lerobot

# Dynamixel SDK 설치
pip install dynamixel_sdk

# Koch 로봇 연결 테스트
python -m lerobot.scripts.find_devices
```

### 기본 제어 예제

```python
from lerobot import KochRobot

# 로봇 초기화
robot = KochRobot()
robot.connect()

# 텔레오퍼레이션 시작
robot.start_teleoperation()
```

Koch v1.1은 Alexander Koch의 오리지널 저비용 로봇 디자인을 개선한 버전입니다. 연구와 학습을 위한 경제적이고 모듈형 설계의 로봇 팔로, LeRobot과 완벽하게 호환됩니다.

<Callout type="tip" title="두 가지 버전">
  Koch 로봇에는 두 가지 버전이 있습니다:
  - **오리지널**: Alexander Koch의 첫 번째 디자인
  - **v1.1**: Jess Moss의 개선된 버전
</Callout>

    - 2x Dynamixel XL430-W250-T 서보 (31,900원)
    - 4x Dynamixel XL330-M288-T 서보 (26,400원)
    - Waveshare 서보 드라이버 보드
    - 12V 전원 공급장치
    - **예상 비용**: 약 20만원 + @
    
    **총 시스템 비용**: 약 50만원

      다만 Dynamixel 서보는 한국에서 구매가 더 저렴할 수 있으니 국내 공급처를 확인해보세요.

## 🛠️ 제작 가이드

### 3D 프린팅 설정

```
• 레이어 높이: 0.2mm
• 노즐 직경: 0.4mm (최대)
• 인필 밀도: ~30%
• 권장 소재: PLA+, ABS, PETG
```

## 🚀 소프트웨어 설정

### LeRobot 통합

```bash
uv pip install -e ".[dynamixel]"
```



### 실제 응용
- 물체 조작 및 픽앤플레이스
- 의류 접기 등 작업
- 교육 및 훈련 플랫폼

### 경량화
- XL330 서보: 단 18g

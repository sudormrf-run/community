# SO-101 Arm

SO-101은 RobotStudio와 Hugging Face가 협력하여 개발한 오픈소스 로봇 팔입니다. LeRobot 라이브러리와 완벽하게 호환되며, AI 로봇공학 학습에 최적화되어 있습니다.

<Callout type="tip" title="최신 버전">
  SO-101은 SO-ARM100의 차세대 버전으로, 개선된 배선과 더 쉽게 조립할 수 있는 설계를 특징으로 합니다.
</Callout>

## 🔧 하드웨어 사양

### 주요 구성품
- **서보 모터**: STS3215 (7.4V)
- **기어비**: 1/345, 1/191, 1/147 선택 가능
- **설계**: 모듈형 설계로 리더/팔로워 구성 가능
- **제작**: 3D 프린팅 가능 (PLA+ 소재)

### 3D 프린팅 설정
```
• 노즐 직경: 0.4mm
• 레이어 높이: 0.2mm
• 인필 밀도: 15%
• 권장 소재: PLA+
```

## 🚀 시작하기

### 1단계: 부품 준비

<Callout type="warning">
  공식 BOM(Bill of Materials)을 확인하여 모든 부품을 준비하세요.
  [공식 GitHub](https://github.com/TheRobotStudio/SO-ARM100)에서 최신 부품 목록을 확인할 수 있습니다.
</Callout>

### 2단계: 3D 프린팅

위의 권장 설정으로 3D 부품을 프린팅합니다. STL 파일은 GitHub 저장소에서 다운로드할 수 있습니다.

### 3단계: 조립

조립 가이드는 [Hugging Face 문서](https://huggingface.co/docs/lerobot)를 참조하세요.

### 4단계: 소프트웨어 설정

## 🎮 활용 예시

### 모방 학습 (Imitation Learning)

```python
# 리더 팔로 원격 조작
from lerobot import SO101Robot

robot = SO101Robot()
robot.connect()

# 데이터 수집 시작
robot.start_recording()
```

### 강화 학습 (Reinforcement Learning)

LeRobot의 다양한 RL 알고리즘을 SO-101에 적용할 수 있습니다.

## 🔌 확장 옵션

### 선택적 업그레이드
- **오버헤드 카메라 마운트**: 작업 공간 전체 뷰
- **손목 카메라**: 정밀한 작업을 위한 근접 뷰
- **촉각 센싱**: 물체 접촉 감지
- **컴플라이언트 그리퍼**: 부드러운 파지
- **베이스 마운트 옵션**: 안정적인 설치

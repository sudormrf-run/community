### 4. 사전 학습 모델 사용하기

HuggingFace에서 제공하는 사전 학습된 모델을 다운로드하고 테스트합니다.

```python
# 사전 학습 모델 다운로드 및 테스트
from lerobot.common.policies.factory import make_policy

# HuggingFace에서 모델 불러오기
policy = make_policy(
    "lerobot/act@main",
    device="cuda"
)

# 모델로 로봇 제어
policy.evaluate()
```

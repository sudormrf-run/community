## 시스템 요구사항

### 필수 요구사항
- **Python 3.10** (3.11 이상은 아직 지원 안 됨)
- **Git**
- **8GB RAM** 이상 권장
- **CUDA 지원 GPU** (선택사항, 학습 시 권장)

## 설치 과정

### 1단계: Miniconda 설치 (권장)

Conda 환경을 사용하면 패키지 충돌을 방지하고 깔끔한 개발 환경을 유지할 수 있습니다.

<Tabs tabs={['macOS', 'Linux', 'Windows']}>
  <div class="tab-panel">
    ```bash
    # Miniconda 다운로드 및 설치
    curl -O https://repo.anaconda.com/miniconda/Miniconda3-latest-MacOSX-x86_64.sh
    bash Miniconda3-latest-MacOSX-x86_64.sh
    
    # 터미널 재시작 후 확인
    conda --version
    ```
  </div>
  <div class="tab-panel">
    ```bash

    bash Miniconda3-latest-Linux-x86_64.sh
    
    # 터미널 재시작 후 확인
    conda --version
    ```
  </div>
  <div class="tab-panel">

    ```powershell
    # https://repo.anaconda.com/miniconda/Miniconda3-latest-Windows-x86_64.exe
    
    # 설치 후 Anaconda Prompt에서 확인
    conda --version
    ```
  </div>
</Tabs>

### 2단계: LeRobot 소스 코드 다운로드

```bash
# LeRobot 저장소 클론
git clone https://github.com/huggingface/lerobot.git
cd lerobot
```

### 3단계: Conda 환경 생성 및 활성화

```bash
# Python 3.10 환경 생성
conda create -y -n lerobot python=3.10

# 환경 활성화
conda activate lerobot
```

<Callout type="tip">
  매번 LeRobot을 사용할 때는 `conda activate lerobot` 명령으로 환경을 활성화해야 합니다.
</Callout>

### 4단계: ffmpeg 설치

영상 처리를 위해 ffmpeg가 필요합니다.

```bash
# Conda를 통한 ffmpeg 설치 (권장)
conda install ffmpeg -c conda-forge
```

### 5단계: LeRobot 설치

```bash
# 기본 설치 (개발 모드)
pip install -e .
```

## 선택적 설치 옵션

### 시뮬레이션 환경

특정 시뮬레이션 환경을 사용하려면 추가 패키지를 설치하세요:

```bash
# ALOHA 시뮬레이션 환경
pip install -e ".[aloha]"

# Push-T 시뮬레이션 환경
pip install -e ".[pusht]"

# 모든 시뮬레이션 환경
pip install -e ".[aloha,pusht,xarm]"
```

### 로봇 하드웨어 드라이버

실제 로봇 하드웨어를 사용한다면 해당 드라이버를 설치하세요:

```bash
# Feetech 모터 (SO-100, SO-101)
pip install -e ".[feetech]"

# Dynamixel 모터
pip install -e ".[dynamixel]"
```

### 실험 추적 도구

모델 학습 과정을 추적하려면 Weights & Biases를 설정하세요:

```bash
# W&B 설치
pip install wandb

# W&B 로그인
wandb login
```

## 문제 해결

### 일반적인 문제와 해결 방법

<details>
<summary>ImportError: No module named 'lerobot'</summary>

LeRobot이 제대로 설치되지 않았습니다. 다음을 확인하세요:
- `conda activate lerobot` 명령으로 환경을 활성화했는지 확인
- `pip install -e .` 명령을 lerobot 폴더 내에서 실행했는지 확인

</details>

<details>
<summary>CUDA 관련 오류</summary>

GPU를 사용하려면 CUDA와 PyTorch가 올바르게 설치되어야 합니다:
```bash
# CUDA 버전 확인
nvidia-smi

# PyTorch 재설치 (CUDA 11.8 예시)
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

</details>

<details>
<summary>ffmpeg 관련 오류</summary>

ffmpeg가 올바르게 설치되지 않았을 수 있습니다:
```bash
# ffmpeg 버전 확인
ffmpeg -version

# conda로 재설치
conda install -c conda-forge ffmpeg=7
```

</details>

## LeRobot이란?

🤗 LeRobot은 HuggingFace에서 개발한 **실제 로봇을 위한 최첨단 머신러닝 플랫폼**입니다. NLP 분야의 Transformers 라이브러리가 혁신을 가져왔듯이, LeRobot은 로봇공학 분야에 동일한 변화를 가져오고자 합니다.

<Callout type="info">
  "LeRobot is to robotics what the Transformers library is to NLP."  
  — Remi Cadene, HuggingFace
</Callout>

## 핵심 목표

LeRobot의 주요 목표는 **로봇공학의 진입 장벽을 낮추는 것**입니다. 이를 통해 누구나 데이터셋과 사전 학습된 모델을 공유하고 활용할 수 있습니다.

## 주요 특징

### 1. 🔧 오픈소스 플랫폼
- PyTorch 기반으로 구축
- 완전한 오픈소스 라이선스
- 활발한 커뮤니티 지원

### 2. 🤖 다양한 하드웨어 지원
- **SO-100/SO-101**: 저가형 로봇 팔
- **Koch v1.1**: 교육용 로봇 키트
- **ALOHA**: 듀얼암 로봇 시스템
- **LeKiwi**: 모바일 로봇 플랫폼

### 3. 🧠 최신 AI 기술
- **Imitation Learning**: 인간 시연을 통한 학습
- **Reinforcement Learning**: 강화학습 알고리즘
- **HIL-SERL**: 인간 개입 강화학습

### 4. 📊 풍부한 리소스
- 사전 학습된 모델 제공
- 다양한 작업별 데이터셋
- 시뮬레이션 환경 지원

## 활용 분야

### 연구 및 개발
- 로봇 학습 알고리즘 연구
- 새로운 로봇 제어 방법 개발
- 벤치마크 및 성능 평가

### 교육
- 로봇공학 입문 교육
- AI와 로봇의 결합 학습
- 실습 중심 커리큘럼

### 산업 응용
- 자동화 시스템 개발
- 로봇 작업 프로그래밍
- 프로토타입 제작

## LeRobot의 장점

1. **접근성**: 복잡한 로봇 프로그래밍 없이 시작 가능
2. **확장성**: 다양한 하드웨어와 작업에 적용 가능
3. **커뮤니티**: HuggingFace 생태계의 강력한 지원
4. **최신 기술**: 최첨단 AI 알고리즘 즉시 활용

## 시작하기 전에

<Callout type="warning" title="필수 요구사항">
  LeRobot을 사용하기 위해서는 Python 3.8 이상과 PyTorch가 필수적으로 설치되어 있어야 합니다.
</Callout>

LeRobot을 시작하기 전에 다음을 준비하세요:

- **Python 3.8** 이상
- **PyTorch** 설치
- 지원되는 로봇 하드웨어 (선택사항)
- 기본적인 Python 프로그래밍 지식

# Imitation Learning 실전 가이드

모방 학습(Imitation Learning)은 인간의 시연을 관찰하여 로봇이 작업을 학습하는 방법입니다. 이 가이드는 LeRobot을 사용한 전체 워크플로우를 단계별로 안내합니다.

<Callout type="tip" title="시작하기 전에">
  이 튜토리얼은 LeRobot이 설치되어 있고 하드웨어가 준비되었다고 가정합니다.
  아직 준비가 안 되었다면 [설치 가이드](/community/docs/physical-ai/lerobot/software/installation)를 먼저 확인하세요.
</Callout>

## 🚀 전체 워크플로우

1. **로봇 캘리브레이션** → 2. **Teleoperation 설정** → 3. **데이터 수집** → 4. **데이터셋 시각화** → 5. **모델 학습** → 6. **평가 및 배포**

## 1️⃣ 로봇 설정 및 캘리브레이션

로봇 하드웨어 설정과 캘리브레이션은 각 하드웨어별 가이드를 참조하세요:

- [SO-ARM101 설정](/community/docs/physical-ai/lerobot/hardware/so-101)
- [Koch v1.1 설정](/community/docs/physical-ai/lerobot/hardware/koch)

## 2️⃣ Teleoperation 설정

원격 조작 설정은 [Teleoperation 가이드](/community/docs/physical-ai/lerobot/software/teleoperation)에서 자세히 다룹니다.

간단한 예시:
```bash
python -m lerobot.teleoperate \
    --robot.type=so101_follower \
    --robot.port=/dev/ttyUSB0
```

## 3️⃣ 데이터 수집

효과적인 데이터 수집 방법은 [데이터 수집 가이드](/community/docs/physical-ai/lerobot/software/data-collection)를 참조하세요.

### 빠른 시작 명령
```bash
python -m lerobot.record \
    --dataset.repo_id=${HF_USER}/my_first_dataset \
    --dataset.num_episodes=50 \
    --robot.type=so101
```

<Callout type="info" title="데이터 수집 팁">
  - 최소 50개 에피소드 수집 (위치당 10개씩)
  - 카메라 위치 고정 유지
  - 물체가 카메라 뷰에 잘 보이도록 설정
  - 간단한 작업부터 시작
</Callout>

## 4️⃣ 데이터셋 시각화

수집한 데이터를 확인하려면:

```bash
# 로컬 시각화 서버 실행
python -m lerobot.serve_dataset \
    --dataset.repo_id=${HF_USER}/my_first_dataset \
    --port=8080
```

또는 [HuggingFace 데이터 시각화 도구](https://huggingface.co/spaces/lerobot/visualize_dataset)를 사용하세요.

## 5️⃣ 모델 학습

모델 학습에 대한 자세한 내용은 [학습 가이드](/community/docs/physical-ai/lerobot/software/training)를 참조하세요.

### ACT 모델 학습 예시
```bash
python -m lerobot.scripts.train \
    --dataset.repo_id=${HF_USER}/my_first_dataset \
    --policy.type=act \
    --output_dir=outputs/my_first_model
```

## 6️⃣ 평가 및 실행

학습된 모델로 로봇 제어:

```bash
python -m lerobot.scripts.eval \
    --robot.type=so101_follower \
    --robot.port=/dev/ttyUSB0 \
    --policy.path=outputs/my_first_model/checkpoint_last.pt \
    --dataset.num_episodes=10
```

## 🎯 실전 프로젝트: 블록 쌓기

### 단계별 진행

1. **환경 준비**
   - 테이블에 3개의 블록 배치
   - 카메라가 전체 작업 공간을 볼 수 있도록 설정

2. **데이터 수집**
   ```bash
   # 블록 쌓기 작업 50회 시연
   python -m lerobot.record \
       --dataset.repo_id=${HF_USER}/block_stacking \
       --dataset.num_episodes=50 \
       --dataset.single_task="Stack 3 blocks"
   ```

3. **모델 학습**
   ```bash
   python -m lerobot.scripts.train \
       --dataset.repo_id=${HF_USER}/block_stacking \
       --policy.type=act \
       --wandb.enable=true
   ```

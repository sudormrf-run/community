# Physical AI 시작하기

Physical AI는 로봇공학과 인공지능을 결합하여 실제 물리적 환경에서 작동하는 지능형 시스템을 구축하는 분야입니다.

## HuggingFace LeRobot 소개

LeRobot은 HuggingFace에서 개발한 오픈소스 로봇 학습 프레임워크입니다.

### 주요 특징
- 🤖 다양한 로봇 플랫폼 지원
- 📚 사전 훈련된 모델 제공
- 🛠️ 쉬운 커스터마이징

## 설치 방법

```bash
# LeRobot 설치
pip install lerobot

# 의존성 패키지
pip install torch torchvision
```

## 첫 번째 프로젝트

간단한 로봇 제어 예제를 통해 Physical AI의 기초를 배워봅시다.

```python
import lerobot

# 로봇 초기화
robot = lerobot.Robot()

# 기본 동작 수행
robot.move_forward(distance=1.0)
robot.turn_left(angle=90)
```
